# Quick Setup Checklist - FB Chemstar Sciences Website

## ✅ Before You Start

- [ ] All 4 files are in the same folder:
  - [ ] `index.html`
  - [ ] `styles.css`
  - [ ] `script.js`
  - [ ] `logo.png` (your company logo)

## ✅ Step 1: Add Your Logo

1. [ ] Find your company logo file (FB Chemstar Sciences logo)
2. [ ] Rename it to **`logo.png`** (important!)
3. [ ] Place it in the same folder as `index.html`
4. [ ] Logo should be at least 60x60 pixels
5. [ ] Recommended: 200x200 pixels or larger

## ✅ Step 2: Add Staff Members

1. [ ] Open `script.js` with a text editor
2. [ ] Find the `STAFF_CREDENTIALS` section at the top
3. [ ] Add your staff members with their names and passwords:

Example:
```javascript
const STAFF_CREDENTIALS = {
    'ali': 'ali_password123',
    'sara': 'sara_password456'
};
```

4. [ ] Save the file (Ctrl+S)

## ✅ Step 3: Test the Website

1. [ ] Open `index.html` in a web browser
2. [ ] Check if the website loads properly
3. [ ] Test the three-dot menu (☰)
4. [ ] Test navigation (Home, About, Services, Contact)
5. [ ] Test Staff Login with your credentials
6. [ ] Verify content editing in Staff Dashboard

## ✅ If Anything is Missing

### Logo Not Showing?
- [ ] Check if `logo.png` is in the same folder as `index.html`
- [ ] Check if the filename is exactly `logo.png` (lowercase)
- [ ] Try refreshing the browser (F5)

### Staff Login Not Working?
- [ ] Make sure you saved `script.js`
- [ ] Check the staff name spelling (case-sensitive)
- [ ] Check the password spelling (case-sensitive)
- [ ] Refresh the browser (F5)
- [ ] Check STAFF_SETUP.md for troubleshooting

### Menu Not Opening?
- [ ] Make sure all three files are in the same folder
- [ ] Refresh the browser (F5)
- [ ] Try a different browser (Chrome, Firefox, Edge)
